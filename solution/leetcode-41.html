Leetcode Solution | 41. First Missing Positive

Link

https://leetcode.com/problems/first-missing-positive/

Logic

  • We first check the smallest number (minN) and biggest number (maxN) in the list.
    • If the smallest is bigger than 1, which means 1 is the first missing positive, then return 1.
    • Or if the biggest is smaller than 1, which means the whole list consists of zeros or negative numbers, then return 1.
  • If the conditions above are not met, the else block will be executed.
    • We then keep the positive numbers and replace the original.
    • If the new biggest number equals the new list length, which means there is no missing positive among the list, return maxN+1.
    • Otherwise, return the smallest missing positive using set.

Python Code

class Solution:
    def firstMissingPositive(self, nums: List[int]) -> int:
        minN = min(nums)
        maxN = max(nums)
        if minN > 1 or maxN < 1:
            return 1
        else:
            nums = list(set([n for n in nums if n > 0]))
            if maxN == len(nums):
                return maxN+1
            else:
                return min(list(set([n for n in range(1, len(nums)+1)]) - set(nums)))
