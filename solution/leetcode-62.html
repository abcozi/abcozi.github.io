Leetcode Solution | 62. Unique Paths

Link

https://leetcode.com/problems/unique-paths/

Logic

  • This is basically a math problem.
  • Let's say m=3, n=7.
  • Then there will be 2 ➡️ (rights) and 6 ⬇️(downs).
  • So the combinations of the actions (right or down) will be 8! / (2! * 6!)

Python Code

class Solution:
    def uniquePaths(self, m: int, n: int) -> int:
        def level(n):
            if n <= 1:
                return 1
            else:
                return n*level(n-1)
        return level(m-1+n-1)//(level(m-1)*level(n-1))
